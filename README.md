# MISTER CODERZ Vault

## Project Vision
MISTER CODERZ Vault is a premium personal digital vault for organizing and accessing digital assets through a modern engineering-first interface.

Telegram is only the storage backend and should remain completely abstracted from the end-user experience.

## Tech Stack
- **Frontend**: React, Vite, Tailwind CSS
- **Backend**: Node.js, Express
- **Other**: Responsive, Mobile-first

## Folder Structure
- `client/`: React frontend application
- `server/`: Node.js Express backend
- `shared/`: Shared resources
- `docs/`: Project documentation

## Installation
1. Clone the repository
2. Install dependencies at the root:
   ```bash
   npm install
   ```
3. Copy `.env.example` to `.env` and configure environment variables.

## Development Commands
To start both the client and server concurrently, run:
```bash
npm run dev
```
